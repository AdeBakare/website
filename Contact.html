1.5 Describing copyright and other constraints affecting websites

Constraints affecting websites is copyright. Copyright is the intellectual property right assigned to its creator. It prevents unauthorised copying and publishing of an original work. Copyright applies to researched data, sharing and reusing data. Therefore any information, data or image obtained will require permission of use from it’s originator, before it is used for a project.

1.6 Describing access issues that need taking into account - describe issues that people with disabilities may face when using web pages and how we can overcome them

Disabilities such as Dyslexia, may cause some issues when visiting web pages. It’s very important to structure web pages very well by not overcomplicating it. Using simple bold visuals will be very helpful. The important factor is you want to create a user friendly web page, that is simple to navigate, simple to read, simple to view and considerate to all, with or without disabilities.

1.7 Describe which are the appropriate file types for websites

When downloading files and viewing websites, there are many file formats. Most are common, others require special programs to open or use. This is a list of file types and their use for website:

  • .AVI — Standard audio/ video file - Used for watching videos
  • .CSS — Cascading StyleSheet file - Used for styling websites
  • .GIF — Graphics Interchange Format file - Used for website images
  • .HTML/ .HTM — HyperText Markup Language file - Used for structuring a website
  • .JPG/ .JPEG — Joint Photographic Experts Group file - Used for website images
  • .MP3 — MPEG Layer 3 sound file - Used for website audio
  • .MPEG/ .MPG — Motion Picture Experts Group file - Used for website videos
  • .PDF — Portable Document Format - Used for website documents
  • .PNG — Portable Network Graphics - Used for website graphics
  • .ZIP — ZIPped file - Used to contain large files on websites
